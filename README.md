# Listas Enlazadas

Proyecto de práctica en Java para resolver retos sobre listas simplemente enlazadas.

## Descripción
En este proyecto se trabajó sobre la implementación de una lista simplemente enlazada, resolviendo 4 retos de lógica y estructuras de datos.

## Retos resueltos
1. Contar cuántas veces aparece un valor dentro de la lista.
2. Limpiar la lista.
3. Invertir la lista.
4. Eliminar duplicados.

## Estructura del proyecto
- `SinglyLinkedList.java`: implementación de la lista simplemente enlazada.
- `LinkedListChallenges.java`: pruebas de los retos resueltos.
- `Main.java`: clase principal para ejecutar el proyecto.

## Cómo ejecutar
1. Abrir el proyecto en Eclipse.
2. Buscar la clase `Main.java`.
3. Ejecutar con **Run As > Java Application**.
4. Verificar en consola los resultados de los 4 retos.
